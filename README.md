### 1. 初始化系统

```shell
$ cd /path/to/project
# 复制模板web入口文件 修改配置环境参数 YII_ENV 值为 local, dev, test, prod
$ cp web/sample.index.php web/index.php

# 复制模板console入口人间 修改配置环境参数 YII_ENV 值为 local, dev, test, prod
$ cd sample.yii yii

```
